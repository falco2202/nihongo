# ビジネス日本語 — フォルダ概要

Tiếng Nhật thực tế dùng hàng ngày tại công ty IT Nhật. Từ email đến Slack, từ họp scrum đến báo cáo tiến độ — tập trung vào các tình huống lặp đi lặp lại nhiều nhất.

## 主なテーマ

| テーマ | 内容 |
|--------|------|
| メール | Viết và đọc email công việc: yêu cầu, xác nhận, báo cáo |
| Slack・チャット | Tin nhắn nội bộ: casual nhưng đúng ngữ cảnh |
| 報連相 | 報告・連絡・相談 — kỹ năng sống còn ở công ty Nhật |
| 会議・朝会 | Scrum, sprint planning, retrospective bằng tiếng Nhật |
| 依頼・断り方 | Cách nhờ vả và từ chối lịch sự |
| 敬語・丁寧語 | Keigo trong context IT: senior, khách hàng, manager |
| 進捗報告 | Báo cáo tiến độ hàng ngày / tuần |

## よく出る語彙（N2/N3）

| 単語 | 読み | 意味（英語） | 意味（ベトナム語） |
|------|------|------------|-----------------|
| 承知しました | しょうちしました | understood, will do | đã hiểu, sẽ làm |
| お世話になっております | おせわになっております | thank you for your continued support | (mở đầu email trang trọng) |
| ご確認ください | ごかくにんください | please confirm/check | vui lòng xác nhận |
| ご連絡いたします | ごれんらくいたします | I will contact you | tôi sẽ liên lạc |
| 取り急ぎ | とりいそぎ | in haste, briefly | vội báo |
| 折り返し | おりかえし | callback, prompt reply | liên lạc lại |
| 検討する | けんとうする | to consider, to review | xem xét |
| 共有する | きょうゆうする | to share (information) | chia sẻ thông tin |
| 調整する | ちょうせいする | to coordinate, to adjust | điều chỉnh, sắp xếp |
| 巻き取る | まきとる | to take over a task | tiếp nhận công việc |

## よく使う表現・テンプレート

### メール書き出し
```
お世話になっております。○○の△△です。
○○の件について、ご連絡いたします。
```

### 依頼する
```
お忙しいところ恐縮ですが、
○○についてご確認いただけますでしょうか。
```

### 進捗報告（Slack）
```
【進捗報告】
✅ 完了: ○○の実装
🔄 作業中: △△のテスト（本日中に完了予定）
⚠️ ブロック: □□について確認待ち
```

### 相談する
```
○○の件でご相談があるのですが、
お時間をいただけますでしょうか。
```

### 断る・調整する
```
ご要望はよく理解できました。
ただ、現在のスケジュール上、○○が難しい状況です。
△△であれば対応可能ですが、いかがでしょうか。
```

### 朝会（スクラム）でよく使う表現
```
昨日は○○を完了しました。
本日は△△を進める予定です。
□□についてブロッカーがあります。
```
