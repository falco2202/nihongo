# 面接・転職 — フォルダ概要

Từ vựng và mẫu câu dùng trong phỏng vấn IT và chuyển việc tại Nhật. Tập trung vào cách mô tả kinh nghiệm kỹ thuật, trả lời câu hỏi hành vi, và thương lượng điều kiện làm việc.

## 主なテーマ

| テーマ | 内容 |
|--------|------|
| 自己紹介 | Tự giới thiệu bản thân và kinh nghiệm kỹ thuật |
| 職務経歴 | Mô tả dự án đã làm, vai trò, thành tích |
| 技術面接 | Trả lời câu hỏi kỹ thuật bằng tiếng Nhật |
| 志望動機 | Lý do ứng tuyển, mục tiêu nghề nghiệp |
| 逆質問 | Câu hỏi ngược lại dành cho nhà tuyển dụng |
| 条件交渉 | Thương lượng lương, remote, điều kiện làm việc |
| 退職・転職理由 | Giải thích lý do nghỉ việc một cách chuyên nghiệp |

## よく使う表現・テンプレート

### 自己紹介
```
私は○年間、主にバックエンド開発を担当してきました。
直近では○○というプロジェクトで、△△の課題に取り組み、
□□という成果を出すことができました。
```

### 志望動機
```
御社を志望した理由は、○○に力を入れている点に魅力を感じたからです。
私のこれまでの経験を活かしながら、さらに成長できる環境だと思いました。
```

### 逆質問
```
入社後、最初の数ヶ月でどのようなことを期待されていますか？
チームの技術スタックは今後どのように変わる予定ですか？
```

### 退職理由（ポジティブに）
```
現職では多くのことを学びましたが、○○の分野でさらに専門性を高めたいと考え、
転職を決意しました。
```
