# ビジネス日本語

このフォルダは、**BJTビジネス日本語能力テストで400点以上を目指すための番号付き練習ノート**です。
単なる文法・語彙の暗記ではなく、実際の職場で日本語を使って「聞く・読む・判断する」力を鍛えることを目的とします。

## 📚 レッスンフォーマット

各レッスンは、基本的に次の流れで作成します。
「読む」だけで終わらせず、BJTで必要な意図理解・状況判断・実際に返す練習までつなげます。

1. **テーマ・目標・レベル**
   - 会議、メール、報連相、依頼、謝罪、交渉、電話対応など
2. **読解文**
   - 約800〜1200字
   - BJTに出やすいビジネス場面を想定
3. **読解問題**
   - 内容理解
   - 意図理解
   - 状況判断
4. **BJT状況判断**
   - 4択問題で「この場面で最も自然な対応」を選ぶ
5. **解答**
   - なぜその答えになるかを確認できる形にする
6. **語彙リスト**
   - 日本語
   - 読み方
   - 英語
   - ベトナム語
7. **文法・表現ポイント**
   - N2/N3レベルを中心に、ビジネスで使いやすい表現を整理する
8. **例文**
   - メール、チャット、報告、相談など、そのまま職場で使える形にする
9. **今日の練習**
   - ベトナム語から自然なビジネス日本語へ変換する

## 🧭 BJT 400向け学習プラン

BJT 400点以上では、単語や文法だけでなく、場面を読んで「相手の意図」「次に取るべき行動」「失礼にならない対応」を選ぶ力が必要です。レッスンは次のように回します。

| フェーズ | 狙い | 主な場面 |
|----------|------|----------|
| 1. 基本連絡 | 誰が何をするかを正確に整理する | メール、報告、相談、依頼 |
| 2. 調整・判断 | 条件、期限、優先順位を自然に確認する | 会議、交渉、断り方、会議後フォロー |
| 3. 社外対応 | 相手への配慮を示しながら事実を伝える | 電話、受付、取引先対応、クレーム対応 |
| 4. 実戦判断 | 曖昧な情報から最適な対応を選ぶ | BJT状況判断、意図理解、未決事項の整理 |

各フェーズでは、ITプロジェクト、営業活動、総務・管理部門、顧客対応、社外イベントなどをローテーションします。同じ表現でも、相手が上司、同僚、他部署、取引先、顧客のどれかによって、丁寧さと情報量を調整します。

## 🧠 BJT対策で意識すること

BJTでは、文法を知っているだけでは十分ではありません。  
次のような力を毎日少しずつ鍛えます。

- 相手の本当の意図を読み取る
- 遠回しな表現を理解する
- 誰が、何を、いつまでに行うのかを整理する
- メールや会議の結論を素早く見つける
- 敬語とビジネスマナーを場面に合わせて使う
- 曖昧な情報から、最も自然な対応を選ぶ

## 📝 メモ

400点以上を目指すには、知識よりも「運用力」が重要です。  
毎日の練習では、知らない単語を覚えるだけでなく、**この場面で相手は何を求めているのか**、**自分ならどう返答するべきか**を考えながら読むことを大切にします。
