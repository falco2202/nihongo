単なる文法・語彙の暗記ではなく、実際の職場で日本語を使って「聞く・読む・判断する」力を鍛えることを目的とします。

## 📅 毎日の練習フォーマット

各レッスンは、基本的に次の流れで作成します。
「読む」だけで終わらせず、BJTで必要な意図理解・状況判断・実際に返す練習までつなげます。

1. **テーマ・目標・レベル**
   - 会議、メール、報連相、依頼、謝罪、交渉、電話対応など
2. **読解文**
   - 約800〜1200字
   - BJTに出やすいビジネス場面を想定
3. **読解問題**
   - 内容理解
   - 意図理解
   - 状況判断
4. **BJT状況判断**
   - 4択問題で「この場面で最も自然な対応」を選ぶ
5. **解答**
   - なぜその答えになるかを確認できる形にする
6. **語彙リスト**
   - 日本語
   - 読み方
   - 英語
   - ベトナム語
7. **文法・表現ポイント**
   - N2/N3レベルを中心に、ビジネスで使いやすい表現を整理する
8. **音声**
   - ある場合はHTMLプレイヤーとMarkdownリンクの両方を置く
9. **例文**
   - メール、チャット、報告、相談など、そのまま職場で使える形にする
10. **今日の練習**
   - ベトナム語から自然なビジネス日本語へ変換する

## 🎧 音声フォーマット

音声を置く場合は、GitHubのMarkdownプレビューでもたどれるように、HTMLプレイヤーと通常リンクを両方置きます。

```html
<audio controls>
  <source src="audio/YYYY-MM-DD.mp3" type="audio/mpeg">
</audio>
```

```markdown
[音声ファイルを開く](audio/YYYY-MM-DD.mp3)
```

MP3ファイルは生成物として扱い、本文側では日付と同じファイル名 `audio/YYYY-MM-DD.mp3` を参照します。

## 🧠 BJT対策で意識すること

BJTでは、文法を知っているだけでは十分ではありません。  
次のような力を毎日少しずつ鍛えます。

- 相手の本当の意図を読み取る
- 遠回しな表現を理解する
- 誰が、何を、いつまでに行うのかを整理する
- メールや会議の結論を素早く見つける
- 敬語とビジネスマナーを場面に合わせて使う
- 曖昧な情報から、最も自然な対応を選ぶ
