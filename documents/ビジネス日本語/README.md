# ビジネス日本語

このフォルダは、**BJTビジネス日本語能力テストで400点以上を目指すための毎日練習ノート**です。  
単なる文法・語彙の暗記ではなく、実際の職場で日本語を使って「聞く・読む・判断する」力を鍛えることを目的とします。

---

## 📅 毎日の練習フォーマット

各 bài học / 記事 は、できるだけ次の形で作成します。

1. **テーマ**
   - 会議、メール、報連相、依頼、謝罪、交渉、電話対応など
2. **読解文**
   - 約800〜1200字
   - BJTに出やすいビジネス場面を想定
3. **読解問題**
   - 内容理解
   - 意図理解
   - 状況判断
4. **解答**
   - なぜその答えになるかを確認できる形にする
5. **語彙リスト**
   - 日本語
   - 読み方
   - 英語
   - ベトナム語
6. **文法・表現ポイント**
   - N2/N3レベルを中心に、ビジネスで使いやすい表現を整理する

## 🧭 BJT対策で意識すること

BJTでは、文法を知っているだけでは十分ではありません。  
次のような力を毎日少しずつ鍛えます。

- 相手の本当の意図を読み取る
- 遠回しな表現を理解する
- 誰が、何を、いつまでに行うのかを整理する
- メールや会議の結論を素早く見つける
- 敬語とビジネスマナーを場面に合わせて使う
- 曖昧な情報から、最も自然な対応を選ぶ

## 📝 メモ

400点以上を目指すには、知識よりも「運用力」が重要です。  
毎日の練習では、知らない単語を覚えるだけでなく、**この場面で相手は何を求めているのか**、**自分ならどう返答するべきか**を考えながら読むことを大切にします。

### 音声の作成

例文の音声を作成する場合は、`OPENAI_API_KEY` を設定してから次のコマンドを実行します。

```bash
python scripts/generate_tts.py ビジネス日本語/2026-05-02.md ビジネス日本語/audio/2026-05-02-report.mp3
```

音声を作成する前に、読み上げ対象のテキストだけを確認したい場合は `--dry-run` を付けます。

GitHub Actions では、`master` ブランチに push されたときに `.github/workflows/build-audio.yml` が実行され、`OPENAI_API_KEY` リポジトリシークレットを使って音声を作成します。
