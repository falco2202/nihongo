# 04 普通文法とビジネス文法

## 目的

ビジネス日本語は、特別な文法だけでできているわけではありません。普通文法を正しく作り、その上で相手への配慮、責任範囲、依頼のやわらかさを足します。

このファイルでは、普通文からビジネス文へ変換する練習をします。

---

## 普通文法の土台

| 機能 | 普通の表現 | 例 |
|------|------------|----|
| 理由 | ので、ため | エラーが出たため、処理を停止しました。 |
| 逆接 | が、しかし | 確認しましたが、原因はまだ分かりません。 |
| 条件 | たら、場合 | 問題がなければ、今日中にリリースします。 |
| 目的 | ように、ために | ミスを防ぐために、チェックリストを使います。 |
| 義務 | なければならない | 本番反映前に承認を取らなければなりません。 |
| 予定 | つもり、予定 | 明日、修正版を提出する予定です。 |

---

## ビジネス文法への変換

| 普通の表現 | ビジネスで自然な表現 | 用途 |
|------------|----------------------|------|
| 確認してください | ご確認いただけますでしょうか | 依頼 |
| 送ってください | お送りいただけますでしょうか | 依頼 |
| できません | 対応いたしかねます | 断り |
| 分かりました | 承知しました | 返答 |
| 今やっています | 現在対応しております | 進捗報告 |
| 終わったら連絡します | 完了次第、ご連絡いたします | 進捗報告 |
| すみません | 申し訳ございません | 謝罪 |
| もう一度教えてください | 再度ご教示いただけますでしょうか | 確認 |
| たぶん問題があります | 問題が発生する恐れがあります | リスク共有 |
| について | につきまして、に関しまして | メール件名・本文 |

---

## よく使うビジネス文法

### 1. 〜ていただけますでしょうか

相手に依頼するときの丁寧な表現です。

- お手すきの際に、ご確認いただけますでしょうか。
- 明日までにレビューしていただけますでしょうか。

### 2. 〜次第

「すぐに〜する」という意味です。報告でよく使います。

- 確認でき次第、ご連絡いたします。
- 修正が完了次第、再度デプロイいたします。

### 3. 〜恐れがあります

リスクをやわらかく、しかし明確に伝える表現です。

- このまま進めると、納期に影響が出る恐れがあります。
- 設定を変更すると、一部の機能に影響する恐れがあります。

### 4. 〜いたしかねます

「できません」を丁寧に断る表現です。

- 申し訳ございませんが、本日中の対応はいたしかねます。
- セキュリティ上の理由により、権限の付与はいたしかねます。

### 5. 〜させていただきます

自分の行動を丁寧に伝える表現です。ただし、使いすぎると不自然になります。

- 本件について、こちらで確認させていただきます。
- 明日の会議で、状況を共有させていただきます。

## 普通文からビジネス文へ

| 普通文 | ビジネス文 |
|--------|------------|
| ログを確認してください。 | ログをご確認いただけますでしょうか。 |
| 終わったら連絡します。 | 完了次第、ご連絡いたします。 |
| 今日中にはできません。 | 申し訳ございませんが、本日中の対応はいたしかねます。 |
| たぶんリスクがあります。 | リスクが発生する恐れがあります。 |
| もう一度説明してください。 | 再度ご説明いただけますでしょうか。 |

## IT職場の例文

### 進捗報告

現在、APIの修正を進めております。確認が完了次第、結果をご連絡いたします。

### リスク共有

この仕様のまま進めると、既存機能に影響が出る恐れがあります。

### 依頼

お手すきの際に、添付の設計書をご確認いただけますでしょうか。

### 断り

申し訳ございませんが、セキュリティ上の理由により、個人アカウントへの権限付与はいたしかねます。

## 練習A: ビジネス文に直す

次の文を、職場で自然な丁寧表現に直してください。

1. この資料を確認してください。
2. 終わったら連絡します。
3. 今日中にはできません。
4. もう一度説明してください。
5. たぶん納期に影響があります。

<details>
<summary>解答例</summary>

1. この資料をご確認いただけますでしょうか。
2. 完了次第、ご連絡いたします。
3. 申し訳ございませんが、本日中の対応はいたしかねます。
4. 再度ご説明いただけますでしょうか。
5. 納期に影響が出る恐れがあります。

</details>

## 練習B: ベトナム語からビジネス日本語へ

1. Khi xác nhận xong, tôi sẽ liên lạc lại.
2. Nếu tiếp tục như vậy, có thể ảnh hưởng đến release.
3. Bạn có thể review tài liệu thiết kế giúp tôi không?
4. Xin lỗi, hôm nay tôi không thể xử lý được.
5. Tôi sẽ kiểm tra vấn đề này ở phía chúng tôi.

<details>
<summary>解答例</summary>

1. 確認が完了次第、ご連絡いたします。
2. このまま進めると、リリースに影響が出る恐れがあります。
3. 設計書をレビューしていただけますでしょうか。
4. 申し訳ございませんが、本日中の対応はいたしかねます。
5. 本件について、こちらで確認させていただきます。

</details>

---

## 注意すること

- 丁寧にしようとして、すべての文に「させていただきます」を入れない。
- 断るときは「できません」だけで終わらせず、理由や代案を添える。
- 報告では「やっています」より「対応しております」のほうが自然。
- リスク共有では、断定しすぎず「恐れがあります」を使うと伝えやすい。
