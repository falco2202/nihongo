# 03 受身・使役・使役受身

## 目的

受身、使役、使役受身は、形だけでなく「誰の視点で話すか」が重要です。まず主語を決めてから、形を選びます。

| 文法 | 基本の意味 | 例 |
|------|------------|----|
| 受身 | AがBに何かをされる | レビューで指摘されました。 |
| 使役 | AがBに何かをさせる | リーダーが新人に確認させました。 |
| 使役受身 | AがBに何かをさせられる | 急に資料を作らされました。 |

---

## 受身

### 形

| 辞書形 | 受身形 |
|--------|--------|
| 書く | 書かれる |
| 読む | 読まれる |
| 話す | 話される |
| 食べる | 食べられる |
| 見る | 見られる |
| する | される |
| 来る | 来られる |

### 基本文型

| 文型 | 例 |
|------|----|
| AはBにVられる | 私は先輩に注意されました。 |
| AはBにNをVられる | 私は先輩にコードをレビューされました。 |
| NがVられる | この機能は多くのユーザーに使われています。 |

### 職場例文

- レビューで変数名を指摘されました。
- このAPIは社内システムで使われています。
- 障害の原因は、運用チームによって調査されました。

---

## 使役

### 形

| 辞書形 | 使役形 |
|--------|--------|
| 書く | 書かせる |
| 読む | 読ませる |
| 話す | 話させる |
| 食べる | 食べさせる |
| 見る | 見させる |
| する | させる |
| 来る | 来させる |

### 基本文型

| 文型 | 例 | メモ |
|------|----|------|
| AはBにNをVさせる | リーダーは新人に手順書を読ませました。 | 他動詞で目的語がある場合 |
| AはBをVさせる | 上司は担当者を会議に参加させました。 | 移動・参加などの自動詞の場合 |
| AはBにVさせてあげる | 先輩は後輩に発表させてあげました。 | 許可・機会を与える |

### 注意

使役は、命令・強制だけではありません。文脈によって「許可する」「機会を与える」という意味にもなります。

- 強制: 上司は新人に資料を作らせました。
- 許可: リーダーは新人に会議で発言させました。

---

## 使役受身

### 形

| 辞書形 | 使役受身形 |
|--------|------------|
| 書く | 書かされる |
| 読む | 読まされる |
| 話す | 話させられる |
| 待つ | 待たされる |
| 食べる | 食べさせられる |
| 見る | 見させられる |
| する | させられる |
| 来る | 来させられる |

### 意味

「自分の意志ではないのに、誰かにやらされた」という負担感があります。

- 急に資料を作らされました。
- 深夜にログを確認させられました。
- 予定外の会議に参加させられました。

ビジネス場面では、直接言うと不満が強く聞こえることがあります。外部向けや上司向けには、やわらかく言い換えるほうが安全です。

| 直接的 | やわらかい表現 |
|--------|----------------|
| 急に資料を作らされました。 | 急きょ資料を作成することになりました。 |
| 深夜に対応させられました。 | 深夜に対応する必要がありました。 |
| 会議に参加させられました。 | 会議に参加することになりました。 |

---

## 比較

| 普通文 | 受身 | 使役 | 使役受身 |
|--------|------|------|----------|
| 先輩が私のコードをレビューしました。 | 私のコードは先輩にレビューされました。 | リーダーが新人にコードをレビューさせました。 | 私はコードをレビューさせられました。 |
| 私がログを確認しました。 | ログが確認されました。 | 上司が私にログを確認させました。 | 私はログを確認させられました。 |
| 担当者が会議に参加しました。 | 担当者は会議に呼ばれました。 | 上司が担当者を会議に参加させました。 | 担当者は会議に参加させられました。 |

## 練習B: 文を変える

次の文を指定された形にしてください。

1. 先輩が私のコードをレビューしました。受身にする。
2. 新人が手順書を読みます。リーダーを主語にして使役にする。
3. 私が深夜対応をします。上司を主語にして使役受身にする。
4. 多くのユーザーがこの機能を使っています。受身にする。
5. 担当者が会議に参加します。マネージャーを主語にして使役にする。

<details>
<summary>解答例</summary>

1. 私のコードは先輩にレビューされました。 / 私は先輩にコードをレビューされました。
2. リーダーが新人に手順書を読ませます。
3. 私は上司に深夜対応をさせられました。
4. この機能は多くのユーザーに使われています。
5. マネージャーが担当者を会議に参加させます。

</details>
