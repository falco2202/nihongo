# 文法トレーニング

このフォルダは、文法を「知っている」状態から、実際の会話・メール・報告で「使える」状態にするための弱点克服ノートです。

対象は、特に混乱しやすい次の項目です。

- 動詞活用
- 自動詞・他動詞
- 受身
- 使役
- 使役受身
- 普通文法からビジネス文法への言い換え

ベトナム語では動詞の形が日本語ほど大きく変化しないため、日本語では「意味」だけでなく「形」を毎日短く反復する必要があります。このフォルダでは、説明を読んだあと、必ず変換練習と職場例文で確認します。

---

## 学習方針

| 方針 | 内容 |
|------|------|
| 形を先に固定する | 意味を考える前に、ます形・ない形・て形・た形をすぐ出せるようにする |
| 助詞で判断する | 自動詞は「が」、他動詞は「を」を中心に確認する |
| 主語を追う | 受身・使役・使役受身では、誰がするか、誰がされるかを先に整理する |
| 普通文から始める | まず普通の日本語で正しく作り、そのあとビジネス表現に直す |
| IT職場で使う | 例文はメール、進捗報告、障害対応、会議、レビューに寄せる |

## 毎日の15分メニュー

| 時間 | 練習 |
|------|------|
| 3分 | 動詞を5個選び、ます形・ない形・て形・た形に変換する |
| 4分 | 自動詞・他動詞を3ペア確認し、助詞「が」「を」を入れる |
| 4分 | 受身・使役・使役受身を1セット作る |
| 4分 | 普通文をビジネス文に言い換える |

## 優先ロードマップ

| 優先 | テーマ | ゴール |
|------|--------|--------|
| 1 | 動詞活用 | 書く、読む、食べる、する、来るを迷わず変換する |
| 2 | 自動詞・他動詞 | 「エラーが出る」「ログを出す」などを正しく使う |
| 3 | 受身 | 「レビューで指摘された」を自然に言える |
| 4 | 使役 | 「新人に確認させる」「担当者を参加させる」を区別する |
| 5 | 使役受身 | 「急に資料を作らされた」のような負担感を理解する |
| 6 | ビジネス文法 | 依頼、報告、謝罪、断りを丁寧に言い換える |

## このフォルダの使い方

1. まず `01_動詞活用トレーニング.md` で形を固める。
2. 次に `02_自動詞・他動詞トレーニング.md` で助詞と意味を確認する。
3. `03_受身・使役・使役受身.md` で「誰が誰に何をしたか」を整理する。
4. `04_普通文法とビジネス文法.md` で職場表現に言い換える。
5. 間違えた文は `99_弱点ログ.md` に残し、翌日の最初に復習する。
