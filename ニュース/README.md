## 記事フォーマット
各記事は以下の構成で書く：
1. **見出し** — タイトル（日本語）
2. **本文** — N2/N3レベルの要約（150〜250字）
3. **語彙リスト** — キーワード5〜8語（読み・意味・英語・ベトナム語）
4. **読解チェック** — 内容確認の質問2〜3問
5. **一言メモ** — 覚えておくべき表現や背景知識
