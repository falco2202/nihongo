# テクニカル — フォルダ概要

このフォルダには、ITエンジニアが日本語で読む・書く技術的な内容についての記事をまとめています。開発現場で実際に使われる日本語の技術表現を中心に学べます。

## 主なテーマ

| テーマ | 内容 |
|--------|------|
| インフラ・クラウド | AWS、GCP、Azureの日本語ドキュメント読解 |
| CI/CD | パイプライン・デプロイに関する日本語表現 |
| セキュリティ | セキュリティ要件・脆弱性の日本語表現 |
| データベース | DB設計・クエリ最適化の日本語 |
| アーキテクチャ | 設計パターン・システム構成の説明 |

## よく出る語彙（N2/N3）

| 単語 | 読み | 意味 |
|------|------|------|
| 構築する | こうちくする | xây dựng / to build, to set up |
| 保守する | ほしゅする | bảo trì / to maintain |
| 障害 | しょうがい | sự cố / failure, incident |
| 脆弱性 | ぜいじゃくせい | lỗ hổng bảo mật / vulnerability |
| 冗長化 | じょうちょうか | dự phòng / redundancy |
| 負荷 | ふか | tải / load |
| 監視する | かんしする | giám sát / to monitor |
| 復旧する | ふっきゅうする | khôi phục / to restore, to recover |
