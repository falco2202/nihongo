# スキル — フォルダ概要

このフォルダには、日本のIT職場で活躍するために必要なスキルについての記事をまとめています。技術的なスキルだけでなく、ビジネス日本語やコミュニケーションスキルも含まれています。

## 主なテーマ

| テーマ | 内容 |
|--------|------|
| ビジネス日本語 | メール、Slack、会議での表現 |
| コードレビュー | レビューコメントの読み書き |
| ドキュメント作成 | 仕様書・設計書の日本語表現 |
| 報連相 | 報告・連絡・相談のやり方 |
| 敬語・丁寧語 | 職場で使う正しい言葉遣い |

## よく出る語彙（N2/N3）

| 単語 | 読み | 意味 |
|------|------|------|
| 仕様書 | しようしょ | tài liệu đặc tả / specification document |
| 課題 | かだい | vấn đề cần giải quyết / issue / task |
| 改善する | かいぜんする | cải thiện / to improve |
| 対応する | たいおうする | xử lý / to handle, to respond |
| 確認する | かくにんする | xác nhận / to confirm |
| 共有する | きょうゆうする | chia sẻ / to share |
| 担当する | たんとうする | phụ trách / to be in charge of |
| 連携する | れんけいする | phối hợp / to collaborate, to integrate |

## 職場でよく使う表現

| 場面 | 表現 | 意味 |
|------|------|------|
| 依頼する | ～をお願いしたいのですが | Tôi muốn nhờ bạn... |
| 確認する | ～について確認させてください | Cho tôi xác nhận về... |
| 報告する | ～の件ですが、～になりました | Về việc..., đã... |
| 相談する | ～について相談があるのですが | Tôi có việc muốn hỏi về... |
