# 学習コレクション

`documents/` の外に置く、全記事横断の復習用フォルダです。語彙と文法を `IT`、`ビジネス`、`生活` の3カテゴリに分けて、1回の復習量を小さくしています。

## フォルダ

| フォルダ | 目的 |
|----------|------|
| [語彙](./語彙/) | 単語・読み・意味メモ・使う文脈・例文をカテゴリ別にカード化する |
| [文法](./文法/) | 文法/表現・使い方・使う文脈・例文をカテゴリ別にカード化する |

## カテゴリ

| カテゴリ | 目安 |
|----------|------|
| IT | 技術、AI、クラウド、セキュリティ、設計、障害対応、開発ニュース |
| ビジネス | メール、会議、報告、相談、交渉、敬語、面接、BJT |
| 生活 | 生活、社会、健康、科学ニュースなど、仕事以外でも使う表現 |

## 更新ルール

新しい記事を `documents/` に追加したら、次も同じタイミングで更新する。

1. 記事に `語彙リスト` と `文法・表現ポイント` を入れる。
2. `powershell -ExecutionPolicy Bypass -File collections/generate-collections.ps1` を実行する。
3. [語彙](./語彙/) と [文法](./文法/) のカテゴリ別ファイルを見て、例文が自然か確認する。
